# 🎵 Music Store Sales Dashboard

## 📌 Project Overview
This project is an interactive Power BI dashboard built to analyze music store sales data. It provides insights into revenue, customers, genres, artists, and sales trends using interactive visualizations.

---

## 📊 Dashboard Features

- 📌 Total Customers
- 💰 Total Revenue
- 🌍 Sales by Country
- 🎵 Sales by Genre
- 📈 Sales Trend
- 👥 Top Customers by Revenue
- ⭐ Top Artists by Sales

---

## 🛠️ Tools & Technologies

- Power BI
- SQL
- PostgreSQL
